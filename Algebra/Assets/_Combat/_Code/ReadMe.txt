Para quitar la base de datos debes seguir los pasos:

1: Quitar la carpeta Plugins y sus contenidos.
2: Quitar los 2 scripts de BasedeDatos.
3: Quitar los respectivos scripts del prefab de la bala y del Scoreboard del canvas
4: Modificar los scripts de Bullet y Scoreboard para retitar los métodos que llamaba a la base de datos
5: Usar los métodos originales sin usar de ambos scripts que sólo usan los playerPref;
